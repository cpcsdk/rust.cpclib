# Fade Generator

!!! info "Command Reference"
    For current usage: `fade --help`  
    Complete documentation: [CLI Help Reference](../CLI_HELP_REFERENCE.md)

Generate smooth color fade tables for Amstrad CPC demos.

## Integration with BndBuild

Fade is available as a standalone `fade` binary but can also be used within [BndBuild](../bndbuild) build scripts using the `fade` command. See [BndBuild Commands](../bndbuild/commands.md#image-management-fade-fade) for integration details.

## Quick Start

For detailed options, use `fade rgb --help`.
