# cpclib-snacli

Command-line tool for manipulating Amstrad CPC snapshot files (.SNA format).

## Installation

```bash
cargo install cpclib-snacli
```

## Usage

```bash
snapshot [OPTIONS] <SUBCOMMAND>
```

See [documentation](https://cpcsdk.github.io/rust.cpclib/snapshot/) for full details.

## Features

- View snapshot information
- Modify snapshot memory
- Extract/inject memory chunks
- Set CPU registers
- Manipulate CRTC state

## Documentation

For detailed examples and usage, visit the [online documentation](https://cpcsdk.github.io/rust.cpclib/snapshot/).
