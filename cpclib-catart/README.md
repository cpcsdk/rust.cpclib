# cpclib-catart

Catart-related functionality for Amstrad CPC demos.

This crate provides utilities for working with Catart demo code, building on top of the BASIC parsing capabilities from `cpclib-basic`.

## Features

- Integration with `cpclib-basic` for BASIC program analysis
- Catart-specific utilities and helpers

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
cpclib-catart = "0.11.0"
```
